STUDIE

Twitter Clone

Dit project is een eenvoudige kloon van Twitter, gebouwd met HTML, CSS, JavaScript, PHP en MySQL. Het doel is om de kernfunctionaliteiten van Twitter na te bootsen, zoals het plaatsen van tweets, het volgen van gebruikers en het liken van berichten.

Source repository

Opendag TCR

Dit project is een interactieve website gemaakt voor de Open Dag van Techniek College Rotterdam. Het is gebouwd met HTML, CSS en JavaScript en biedt gedetailleerde informatie over de school, opleidingen en het programma van de Open Dag.
